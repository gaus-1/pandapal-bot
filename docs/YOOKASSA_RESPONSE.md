PandaPal — это образовательная платформа, которая помогает детям получать помощь с домашними заданиями и обучением через Telegram-бота с искусственным интеллектом.

### Основные функции:
- Помощь с домашними заданиями по всем школьным предметам
- Объяснение сложных тем простым языком
- Анализ изображений задач
- Обработка голосовых сообщений
- Безопасная модерация контента для детей
