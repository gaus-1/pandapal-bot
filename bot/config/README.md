# Config - Конфигурация

Настройки приложения, промпты для AI и паттерны модерации.

## Файлы

- `settings.py` - настройки из переменных окружения (Pydantic Settings)
- `prompts.py` - системные промпты для YandexGPT
- `forbidden_patterns.py` - паттерны для модерации контента

## Settings

Настройки загружаются из `.env` через Pydantic:

```python
from bot.config.settings import settings

database_url = settings.database_url
telegram_token = settings.telegram_bot_token
```

Все настройки валидируются при старте приложения.

## Промпты

Системные промпты для AI определяют поведение бота:
- Адаптация под возраст
- Стиль общения
- Поддержка предметов
- Ограничения и запреты

## Паттерны модерации

Паттерны для фильтрации опасного контента:
- Запрещенные слова и фразы
- Паттерны для разных языков (RU, EN, DE, FR, ES)
- Категории контента

## Важно

- Не коммитить секреты в код
- Все секреты только в `.env`
- Промпты можно менять без перезапуска (если загружаются динамически)
