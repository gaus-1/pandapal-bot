# Config - Конфигурационные файлы

Файлы настройки проекта.

## Файлы

- `alembic.ini` - конфигурация миграций базы данных Alembic
- `pytest.ini` - настройки тестового фреймворка pytest
- `pyproject.toml` - метаданные проекта Python
- `pyrightconfig.json` - настройки статического анализатора Pyright
- `env.template` - шаблон переменных окружения для локальной разработки

## Важно

Реальный `.env` файл НЕ в этой папке. Он остается в корне проекта и игнорируется git.
