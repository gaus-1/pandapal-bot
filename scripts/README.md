# Scripts - Утилиты

Вспомогательные скрипты для обслуживания проекта.

## Основные скрипты

- `check_db.py` - проверка подключения к базе данных PostgreSQL
- `init_db_simple.py` - простая инициализация БД для локальной разработки
- `check_circular_imports.py` - поиск циклических импортов в коде
- `check_dead_code.py` - обнаружение неиспользуемого кода через Vulture
- `fix_encoding.py` - автоматическое исправление кодировки файлов на UTF-8
- `check_dependencies.py` - проверка версий зависимостей
- `update_requirements_txt.py` - обновление requirements.txt
- `view_database.py` - просмотр данных в БД
- `export_metrics.py` - экспорт метрик для анализа

Все скрипты запускаются из корня проекта.
