# SQL - Скрипты для базы данных

SQL скрипты для создания структуры базы данных PostgreSQL.

## Файлы

- `01_drop_all_tables.sql` - удаление всех таблиц
- `02_create_tables.sql` - создание всех таблиц
- `03_add_premium_subscriptions.sql` - добавление Premium подписок
- `04_add_payment_method_fields.sql` - поля для платежей
- `05_create_payments_table.sql` - таблица платежей
- `06_create_daily_request_counts.sql` - счетчики запросов

## Использование

Эти скрипты используются для первоначальной настройки БД. В продакшене применяются миграции Alembic.

## Важно

- Всегда делай бэкап перед применением
- Проверяй скрипты на тестовой БД
- Не изменяй скрипты после применения в продакшене
